# セキュリティ監査ガイド - 監査概要と日常チェックリスト

## 🔍 監査の概要

### 監査の目的
- **患者情報保護の確保**: 個人情報漏洩リスクの最小化
- **法規制遵守の確認**: 関連法令・ガイドラインへの適合性確認
- **セキュリティ体制の評価**: 現状の安全管理措置の有効性検証
- **継続的改善**: リスク軽減と運用改善のための課題特定

### 監査の種類
- **自己監査**: 院内での定期的な自主点検
- **内部監査**: 医療機関内の監査部門による監査
- **外部監査**: 第三者機関による客観的な監査
- **規制当局監査**: 行政機関による法的監査
